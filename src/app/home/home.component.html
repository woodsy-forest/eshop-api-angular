How to set up the online shop
    Use the Account.Register Api to register to the shop.
    Create a Email Confimation Page and set the link in Setting.Name=confirmationEmail.confirmationLink
    In the Email Confirmation Page, get the CustomerId from the Url and call Account.Email-Confirmation Api
    In case of Reset Password, create a Reset Password Page and set the link in Setting.Name=resetPassword.resetLink
    In the Reset Password Page, get the token from the Url and call Account.Reset-Password Api
    Add some categories using Category Api
    Add some attributes (if any) using Attribute Api
    Add some discounts (if any) using Discount Api
    Config the settings using Setting Api
    Add some products using Product Api
    Add To Cart
